Propriedades customizadas

Também conhecidas como variáveis css (custom properties), permite definirmos valores no CSS que podem ser reutilizados no nosso código. A propriedade é herdada pelos elementos filhos. É comum definirmos as mesmas nos elementos: root ou html, assim teremos acesso à propriedade em todos os elementos do site.

  • --roxo: #caf

    Define uma propriedade customizada.

  • var(--roxo)

    A função var utiliza uma propriedade customizada.
